# DynmapSync
Merge your Dynmap from multiple servers into one.
The project is still in development and not ready for production use.

| Dynmap Feature         | Supported    |
|------------------------|--------------|
| Player Sync            | ✅            |
| Multi World view       | ✅            |
| Single World view      | ✅            |
| Customizable Scheduler | ✅            |
| Chat                   | 🛠️ Readonly |
| Auth                   | ❌            |
| Player Join/Quit       | 🛠️          |
| Markers                | 🛠️          |
| Marker Sets            | 🛠️          |
| Area Marker            | 🛠️          |
| Circle Marker          | 🛠️          |
| Poly Line Marker       | 🛠️          |
| Marker Update          | ❌            |
| Day/Night              | 🛠️          |
| Weather                | 🛠️          |
| Flat view              | 🛠️          |
| Surface view           | ❌ Scheduled  |
| FileTree Storage       | ✅            |
| MySQL Storage          | ❌            |
| SQLite Storage         | ❌            |
| S3 Storage             | ❌            |

## Legend
✅ = Supported
🛠️ = In Development
❌ = Not Supported
