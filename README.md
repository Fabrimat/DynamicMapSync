Merge your Dynmap from multiple servers into one.

| Dynmap Feature    | Supported    |
|-------------------|--------------|
| Player Sync       | 🛠️          |
| Multi World view  | 🛠️          |
| Single World view | 🛠️          |
| Chat              | 🛠️ Readonly |
| Auth              | ❌            |
| Player Join/Quit  | 🛠️          |
| Markers           | 🛠️          |
| Marker Sets       | 🛠️          |
| Area Marker       | 🛠️          |
| Circle Marker     | 🛠️          |
| Poly Line Marker  | 🛠️          |
| Day/Night         | 🛠️          |
| Weather           | 🛠️          |
| Flat view         | 🛠️          |
| Surface view      | Scheduled    |
| FileTree Storage  | ✅            |
| MySQL Storage     | ❌            |
| SQLite Storage    | ❌            |
| S3 Storage        | ❌            |
